# DJ Game / Quiz Master 🎮

## Descrição

O **DJ Game** é um jogo interativo em formato de **quiz 2D baseado na web**, desenvolvido no âmbito de um **trabalho académico**.

O principal objetivo do projeto foi explorar o **uso de Inteligência Artificial (IA) como ferramenta de apoio ao desenvolvimento de software**, demonstrando como estas tecnologias podem auxiliar na criação de aplicações interativas.

O jogo permite testar conhecimentos através de um sistema de perguntas e respostas, incluindo funcionalidades como temporizador, sistema de pontuação e leaderboard.

---

## Objetivo do Projeto

Este projeto teve como objetivos principais:

- Explorar o uso de **IA no desenvolvimento de aplicações**
- Desenvolver um **jogo de quiz interativo**
- Demonstrar como ferramentas de IA podem ajudar na:
  - geração de código
  - estruturação de funcionalidades
  - desenvolvimento de interfaces
- Aplicar conceitos de **desenvolvimento web frontend**

---

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** – Estrutura da aplicação
- **CSS3** – Estilização e layout
- **JavaScript** – Lógica do jogo
- **Bootstrap 5** – Componentes de interface e responsividade
- **Git / GitHub** – Controlo de versões

---

## Funcionalidades

O jogo inclui várias funcionalidades interativas, tais como:

- Sistema de **perguntas e respostas**
- **Temporizador por pergunta**
- **Contador de pontuação**
- **Ecrã de resultados**
- **Histórico de jogos**
- **Leaderboard (ranking de jogadores)**
- **Modo de gestão de perguntas (backoffice)**
- Possibilidade de **importação de perguntas via API**
- **Tema escuro e claro**
- Interface com animações e micro-interações

## Como Executar o Projeto

1. Clonar o repositório:

```bash
git clone https://github.com/Xkim2000/dj-game.git
```

2. Entrar na pasta do projeto:

 ```bash
cd dj-game
```

3. Abrir o ficheiro index.html no browser.

Não é necessário instalar dependências adicionais, pois o projeto funciona diretamente no navegador.
